\subsection{Detailed Description}
XML text. 

A text node can have 2 ways to output the next. \char`\"{}normal\char`\"{} output and CDATA. It will default to the mode it was parsed from the XML file and you generally want to leave it alone, but you can change the output mode with \doxyref{Set\-CDATA()}{p.}{classpandora_1_1TiXmlText_de35866aa708cc2ae73fcb9b786653f3} and query it with \doxyref{CDATA()}{p.}{classpandora_1_1TiXmlText_21f317b8bb9d4af409698be1af20045d}.
