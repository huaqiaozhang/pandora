\subsection{Detailed Description}
The parent class for everything in the Document Object Model. 

(Except for attributes). Nodes have siblings, a parent, and children. A node can be in a document, or stand on its own. The type of a \doxyref{Ti\-Xml\-Node}{p.}{classpandora_1_1TiXmlNode} can be queried, and it can be cast to its more defined type.

Every node in the XML tree will be conditionally visited and the host will be called back via the \doxyref{Ti\-Xml\-Visitor}{p.}{classpandora_1_1TiXmlVisitor} interface.

This is essentially a SAX interface for Tiny\-XML. (Note however it doesn't re-parse the XML for the callbacks, so the performance of Tiny\-XML is unchanged by using this interface versus any other.)
